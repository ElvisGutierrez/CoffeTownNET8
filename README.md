CoffeeTown – Sistema de Pedidos para Cafetería

Aplicación web desarrollada para gestionar pedidos, ventas, usuarios y administración de productos y categorías dentro de una cafetería. Incluye módulos de autenticación, roles (usuario/admin), compra en línea y panel administrativo completo.
